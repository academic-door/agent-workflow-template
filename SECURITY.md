# Security Policy

## Sensitive Data

Do not open Issues or Pull Requests containing:

- API keys or tokens
- passwords or cookies
- private datasets
- local file paths that reveal personal information
- unpublished research data

## Agent Access

Grant agents access only to the repositories they need.

Prefer selected-repository authorization over account-wide or organization-wide access.

## Reporting Issues

If you find a privacy or security problem in this template, open a private security advisory or contact the maintainer through a non-public channel.
